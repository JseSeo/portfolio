재능대학교 캡스톤 홈페이지

프로젝트 개요

매년 교내에서 진행되어 전시되고 사라지는 캡스톤 프로젝트 결과물을 캡스톤 홈페이지에 전시함으로써 더 많은 전시 및 홍보 기회와 후배님들께 동기부여가 되고자 했습니다.

본 프로젝트는 AWS 클라우드를 기반으로 설계된 캡스톤 프로젝트 관리 웹사이트로, 프론트엔드와 백엔드가 분리된 구조를 채택하였습니다. 사용자는 웹사이트를 통해 다양한 프로젝트 정보를 등록하고, 조회 및 관리할 수 있습니다.

프로젝트 사이트: 캡스톤 프로젝트 홈페이지 바로가기

프로젝트 시연 영상

아키텍처 상세 설명

프론트엔드 (Front-End)

  • HTML, CSS, JavaScript로 구현된 사용자 인터페이스(UI)와 사용자 경험(UX)
  • AWS S3를 활용한 정적 파일 호스팅
  • 반응형 디자인을 통한 다양한 디바이스 지원

백엔드 (Back-End)

  • AWS EC2에 배포된 Spring Framework 기반 개발
  • Spring Data JPA를 통한 객체지향적 데이터베이스 접근
  • 인증과 권한 관리를 포함한 RESTful API 제공
  • RestDocs를 활용한 API 문서화

데이터베이스 (Database)

  • AWS RDS의 MySQL 데이터베이스 활용
  • 데이터 백업과 복원, 보안 및 확장성 제공
  • 프로젝트 데이터의 신뢰성 보장

AWS 클라우드 활용

EC2

백엔드 서버 호스팅 및 확장 가능한 환경 운영

S3

정적 파일 배포를 통한 빠르고 안정적인 콘텐츠 제공

RDS

관계형 데이터베이스 운영으로 높은 가용성과 성능 보장

아키텍처의 주요 특징

  • 확장성: AWS 클라우드 인프라를 활용한 트래픽 대응
  • 유지보수성: Spring Data JPA와 RestDocs를 통한 효율적인 관리
  • 안정성: AWS RDS와 EC2를 통한 안정적인 서비스 운영

기대 효과

본 아키텍처를 통해 사용자는 쉽고 빠르게 프로젝트를 관리할 수 있으며, AWS의 강력한 클라우드 인프라를 활용하여 고성능, 고가용성의 웹 서비스를 제공할 수 있습니다.
