재능대학교 캡스톤 홈페이지

프로젝트 개요

매년 교내에서 진행되어 전시되고 사라지는 캡스톤 프로젝트 결과물을 캡스톤 홈페이지에 전시함으로써 더 많은 전시 및 홍보 기회와 후배님들께 동기부여가 되고자 했습니다.

본 프로젝트는 AWS 클라우드를 기반으로 설계된 캡스톤 프로젝트 관리 웹사이트로, 프론트엔드와 백엔드가 분리된 구조를 채택하였습니다. 사용자는 웹사이트를 통해 다양한 프로젝트 정보를 등록하고, 조회 및 관리할 수 있습니다.


프로젝트 사이트: 캡스톤 프로젝트 홈페이지 바로가기

프로젝트 시연 영상

아키텍처 상세 설명

본 프로젝트는 CI/CD 파이프라인, 컨테이너 기반 배포, 그리고 보안 인증 시스템을 중심으로 설계된 마이크로서비스 아키텍처입니다.

주요 구성 요소

  • CI/CD 파이프라인: Jenkins를 통해 Docker 이미지를 빌드하고, AWS ECR에 업로드하여 지속적인 통합과 배포를 자동화합니다.
  • Blue/Green 배포: AWS ECS와 ALB를 활용해 무중단 배포를 지원합니다.
  • 프론트엔드: HTML, CSS, JavaScript로 개발되었으며 AWS S3를 통해 정적 파일을 호스팅합니다.
  • 백엔드: Spring Framework 기반으로 개발되었으며 RESTful API 제공과 Spring Security로 인증/권한 관리를 수행합니다.
  • 데이터베이스: AWS RDS MySQL을 사용하여 프로젝트 데이터를 안전하게 저장하고 관리합니다.
  • 파일 저장소: 파일 업로드 및 관리는 AWS S3를 활용하여 효율적으로 처리합니다.

AWS 클라우드 서비스

EC2

Spring Boot 백엔드 애플리케이션 배포 및 실행

S3

프론트엔드 정적 파일 배포

RDS

데이터베이스 운영 및 관리

ECR

Docker 이미지를 저장하는 레지스트리

ECS

컨테이너 기반 백엔드 배포와 운영

ALB

트래픽 분산 및 Blue/Green 배포 지원

아키텍처의 주요 특징

  • 확장성: AWS 클라우드를 활용해 높은 트래픽에도 대응 가능한 인프라 구축
  • 유지보수성: Spring Data JPA와 RestDocs를 통해 코드 관리와 API 문서화 지원
  • 안정성: AWS RDS와 ECS를 통한 안정적이고 신뢰성 있는 서비스 운영

기대 효과

이 프로젝트를 통해 학생들의 캡스톤 프로젝트가 지속적으로 홍보되며, AWS 클라우드 인프라를 활용한 빠르고 안정적인 웹 서비스를 제공할 수 있습니다.
